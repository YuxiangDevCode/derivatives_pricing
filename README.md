# Equity Derivatives Project

Pricing models and volatility surface calibration.

## Black-Scholes Module
- Analytic pricing for European options
- Closed-form Greeks (Delta, Gamma, Vega, Theta, Rho)
- Modular implementation with unit tests

## Monte Carlo Module
- Vectorized GBM path simulation
- European option pricing (calls, puts)
- Confidence interval estimation
- MC vs Black-Scholes comparison
- Convergence analysis with visualization
